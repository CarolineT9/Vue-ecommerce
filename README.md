# Vue 3 E-commerce

Loja Online

## Entidades:


### Produto

- id
- nome
- descrição
- preço
- imagem

### Categoria

- nome
- descrição

### Carrinho de compras
- produtos

## Componentes

### ProductCard
### ShoppingCart

## Páginas

- / -> Todos os produtos
- /categoria/5 -> Protdutos da categoria 5
- /cart -> Ver o Carrinho de compras


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
